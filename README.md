# Client-Side API Web App

This project is a browser-based web application that fetches data from free public APIs, including:

- Rest Countries API
- Exchange Rate API
- Wikipedia REST Summary
- World Bank Indicators

The application runs entirely on the client side using JavaScript and does not require a backend server.

## Features

- Search countries by name and view their stats and charts
- Compare up to 3 countries
- Leaderboard with different filters
- Interactive map visualization
